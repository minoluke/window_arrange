# 個展配置シミュレーター

ガラス窓の絵を複数組み合わせた個展の会場レイアウトを事前にテストするローカルアプリ。
`index.html` をブラウザで開くだけで動作する。インストール不要。

---

## 使い方

1. `index.html` をブラウザで開く
2. 左サイドバーで壁サイズを設定
3. 「絵を追加」で新しい絵を追加
4. 絵をクリックして選択 → サイドバーで各種パラメータを調整
5. 絵をドラッグして壁上の好きな位置に配置

---

## 設定できる変数

### グローバル（壁）
| 項目 | 説明 | デフォルト |
|------|------|-----------|
| 壁の幅 | ギャラリーの壁の横幅 (cm) | 500cm |
| 壁の高さ | ギャラリーの壁の高さ (cm) | 300cm |

### 絵ごとの設定（各絵で独立）
| 項目 | 説明 | デフォルト |
|------|------|-----------|
| 列数 (cols) | ガラスブロックの横の数 | 3 |
| 行数 (rows) | ガラスブロックの縦の数 | 4 |
| ブロック幅 | 1個のガラスブロックの幅 (cm) | 15cm |
| ブロック高 | 1個のガラスブロックの高さ (cm) | 20cm |
| 縁の太さ | 仕切り・外枠バーの太さ (cm) | 3cm |
| 額縁幅 | 額縁（ピクチャーフレーム）の幅 (cm) | 5cm |

---

## 絵のサイズ計算式

```
総幅  = 2 × 額縁幅 + (列数 + 1) × 縁の太さ + 列数 × ブロック幅
総高  = 2 × 額縁幅 + (行数 + 1) × 縁の太さ + 行数 × ブロック高
```

---

## 技術仕様

- シングルファイル構成（`index.html` のみ）
- 純粋な HTML / CSS / JavaScript（フレームワーク・ビルドツール不使用）
- ガラス窓グリッドは HTML Canvas API で描画
- 絵の配置は `position: absolute` の DOM 要素 + マウスイベントでドラッグ実現
- 座標の正規単位は cm。表示時のみ px に変換（スケール自動計算）
- ウィンドウリサイズ時にスケールを再計算、cm座標は保持
